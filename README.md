📌 Cadastro de Alunos

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de JavaScript, focando na manipulação do DOM, validações e gerenciamento de dados em memória.

A aplicação simula um sistema simples de cadastro de alunos, permitindo operações básicas de um CRUD.

🚀 Funcionalidades

O sistema permite:

➕ Cadastrar novos alunos
📋 Listar alunos dinamicamente em uma tabela
✏️ Editar dados de um aluno
❌ Excluir alunos cadastrados
🚫 Impedir cadastro de nomes duplicados (ignorando maiúsculas/minúsculas)
🧠 Conceitos aplicados

Durante o desenvolvimento, foram utilizados:

Manipulação do DOM (getElementById, innerHTML)
Eventos (onclick)
Estruturas condicionais (if, operadores lógicos)
Arrays e métodos:
push
splice
forEach
some (validação de duplicidade)
Tratamento de strings:
toLowerCase() (comparação sem case sensitive)
Template strings (` `)
Controle de estado (lista de alunos)
⚙️ Como funciona
O usuário preenche os campos (nome, idade e curso)
O sistema valida se todos os campos foram preenchidos
Verifica se já existe um aluno com o mesmo nome (ignorando maiúsculas/minúsculas)
Caso válido, o aluno é adicionado ao array
A tabela é atualizada dinamicamente
Cada aluno possui opções de edição e exclusão
🎯 Objetivo do projeto

O principal objetivo foi fortalecer a base em JavaScript, trabalhando com:

manipulação de dados em arrays
validações de formulário
atualização dinâmica da interface
lógica de controle de dados
